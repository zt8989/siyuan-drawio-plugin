
# Siyuan Draw.io Plugin

[中文](README_zh_CN.md) | [English](README.md)

## Description
Integrated into Siyuan as a plugin

### Embedding into Siyuan                                     
                                                              
1. In drawio, go to File -> Embed -> IFrame -> Create -> Copy 
2. Paste into Siyuan document
